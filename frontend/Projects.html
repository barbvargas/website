Wine Quality Testing

We aimed to predict and classify wine quality using a neural network, logistic regression, and random forest classifier. The dataset we used can be found on the UCI Machine Learning repository and consists of two csv files, one white wine and the other red, each with a list of features and the wine quality rating. After cleaning, training and testing we were able to determine the random forest classifier resulted in the highest accuracy with an F1 accuracy score of 0.99. Github Link
